O áudio fica salvo neste aparelho. A transcrição é feita depois, na aba Entrevistas.
Critério de qualidade: grave no mínimo 5 minutos de fala para validar a transcrição. O app avisa se a duração ficar abaixo disso.
Obtida em console.groq.com → API Keys. A chave fica salva apenas neste navegador, neste aparelho. Nunca é enviada a outro lugar além da API do Groq.
Transcrição: Groq · Whisper large-v3 · idioma fixado em português.
Áudios longos são divididos em blocos de ~10 minutos e as transcrições são unidas na ordem correta.
Dados (áudio e transcrições) armazenados localmente via IndexedDB — excluir uma entrevista remove tudo de forma definitiva.
Diarização automática não disponível no Groq: use os rótulos de falante manuais em cada trecho (toque no rótulo para alternar).
